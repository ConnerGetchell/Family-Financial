Hello and Welcome to Family Financial!
     Thank you for downloading.

This Read Me will show you how to run
the project locally.

1. Download the zip to your machine.

2. Place the folder in a "Project"
   folder of your choice.

3. Install the requirements:
   pip install -r requirements.txt

4. Create the database:
   py manage.py migrate

5. Run the server:
   py manage.py runserver

6. Open your web-browser of choice to
   127.0.0.1:8000.

-----------------Notes----------------

*Admin: 127.0.0.1:8000/admin/
User: admin
Pass: admin

*Important: Admin can view the site,
but the admin cannont post from the
site. The admin must be in the admin
console defined above.
